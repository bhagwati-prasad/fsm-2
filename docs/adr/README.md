# Architecture Decision Records (ADRs)

This directory stores architecture decisions for the FSM tool.

## Status Legend

- `proposed`: decision is under discussion
- `accepted`: decision is approved and active
- `superseded`: decision has been replaced by a newer ADR
- `deprecated`: decision is no longer preferred but may still exist

## How to add a new ADR

1. Copy [TEMPLATE.md](./TEMPLATE.md)
2. Name it with the next sequence number, e.g. `0002-some-decision.md`
3. Fill in context, decision, consequences, and alternatives
4. Add an entry to the ADR Index table above
